หน่วยแบบอัตรส่วน

em = การอ้างอิงขนาด element parent ที่ใกล้ที่สุด ใช้ในการดำเนินการจำนวนเท่า
เช่น หากใช้ 10px 2em คือ 2 เท่าของขนาด 20px
1.4em=1.4px
1em = 10px

rem กำหนดขนาดโดยอ้างอิงกับ root element ปกติ font-size จะอยู่ที่ 16px

vw 1% หรือ 1/100 ของ viewport width - width ของ browser viewport เท่ากับ 750px ค่า 1vw =7.5px

vh 1% หรือ 1/100 ของ viewport height เท่ากับ 900px ค่า 1vh = 96px

vmin,vmax กำหนดค่าต่ำสุด และค่าสูงสุดของ viewport
